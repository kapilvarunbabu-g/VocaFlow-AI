# 🎙 VocaFlow AI

> Read less. Understand more. Learn at the speed of thought.

VocaFlow AI is an advanced AI-powered voice learning platform inspired by modern productivity and accessibility tools like Speechify. It transforms documents into immersive learning experiences using synchronized text-to-speech, AI-generated flashcards, quizzes, podcasts, mind maps, and accessibility-focused reading tools.

---

# 🚀 Features

## 📖 AI Reader
- Real-time text-to-speech reading
- Word-by-word highlighting
- Smart reading synchronization
- Multi-speed playback

## 🧠 AI Learning System
- AI-generated flashcards
- Interactive quizzes
- Smart summaries
- Key point extraction
- Mind map generation

## 🎙 AI Studio
- Podcast generator
- AI voice cloning
- Text-to-speech studio
- AI dubbing & translation

## ♿ Accessibility Features

- ADHD focus mode
- Adjustable font size
- Adjustable line spacing
- High contrast support

## 🎨 UI/UX
- Modern glassmorphism design
- Dark & light theme
- Responsive mobile design
- Animated waveforms
- Interactive AI assistant
